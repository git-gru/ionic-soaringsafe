The internet will shut off at bedtime. You can always do this later. More Info

At bedtime the internet connection to the device will be shut off.

This means that the user can still use non internet connected apps such as games. The ability to block non internet apps and games is coming in a future release.

Note that for safety & emergency reasons, calls and texting cannot be blocked. If texting late at night is a problem we recommend taking the device at night or placing it in a safe areay.
